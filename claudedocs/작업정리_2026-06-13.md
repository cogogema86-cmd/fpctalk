## 2. 🔴 채팅 7일 멈춤 문제 — 해결 완료

### 원인
- **Supabase 무료 플랜은 7일간 아무도 안 쓰면 DB를 자동 정지** → 앱 전체(로그인·채팅) 멈춤.
- 더 깊은 원인: 매일 도는 정리 cron이 있었는데도 멈춘 이유 = **시스템이 cron 호출을 로그인 페이지로 튕겨내서 DB를 한 번도 못 깨우고 있었음.** (이번에 수정함)

### 해결 (이중 안전망)
매일 자동으로 DB를 깨워서 7일 정지를 막습니다:
| 깨우는 주체 | 시각(한국) |
|---|---|
| Vercel 자동 | 매일 **오전 9시** |
| cron-job.org (외부) | 매일 **오후 3시** ← 사장님이 직접 등록 완료 |

→ 둘 중 하나가 멈춰도 다른 하나가 막아줌. **이제 7일 정지 안 됨.**

### 사장님이 가끔 봐주면 좋은 것
- cron-job.org → Cronjobs → 해당 작업 → **History** 탭에 매일 ✅ Success 쌓이는지 확인.
- 빨간 실패가 보이면 알려주기.
- 점검용 주소(눌러서 `{"ok":true...}` 나오면 정상): https://www.fpctalk.com/api/cron/keepalive
